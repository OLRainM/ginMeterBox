# 📖 使用指南

## 快速开始

### 方法1：使用启动脚本（推荐）

双击 `start.bat` 文件即可启动服务器。

### 方法2：命令行启动

```bash
# 进入项目目录
cd e:\project\go-ele

# 安装依赖（首次运行）
go mod tidy

# 启动服务器
go run main.go
```

### 方法3：编译后运行

```bash
# 双击 build.bat 或运行以下命令
go build -o bin\go-ele.exe main.go

# 运行编译后的程序
cd bin
go-ele.exe
```

## 访问系统

启动服务器后，在浏览器中打开：

```
http://localhost:8080
```

## 基本操作

### 1️⃣ 设置价格

在页面顶部的"价格配置"区域设置：
- **水价**：每吨水的价格（默认 4.3 元）
- **电价**：每度电的价格（默认 0.72 元）

### 2️⃣ 添加账单记录

1. 点击 **"➕ 新增记录"** 按钮
2. 填写表单信息：
   - **住户编号**：如 "101"、"A1" 等
   - **缴费月份**：选择月份
   - **本月水表读数**：当前月份的水表读数
   - **上月水表读数**：上个月的水表读数
   - **水表补差**：如有误差需要调整，填写补差值（可选）
   - **本月电表读数**：当前月份的电表读数
   - **上月电表读数**：上个月的电表读数
   - **电表补差**：如有误差需要调整，填写补差值（可选）
   - **管理费**：物业管理费（可选）
3. 点击 **"保存"**

系统会自动计算：
- ✅ 用水量 = 本月读数 - 上月读数 + 补差
- ✅ 用电量 = 本月读数 - 上月读数 + 补差
- ✅ 水费 = 用水量 × 水单价
- ✅ 电费 = 用电量 × 电单价
- ✅ 总费用 = 水费 + 电费 + 管理费

### 3️⃣ 编辑记录

- 在表格中找到要修改的记录
- 点击 **"编辑"** 按钮
- 修改信息后点击 **"保存"**

### 4️⃣ 删除记录

- 在表格中找到要删除的记录
- 点击 **"删除"** 按钮
- 确认删除操作

### 5️⃣ 筛选查询

- 使用 **"按月份筛选"** 下拉框
- 选择特定月份查看该月的所有记录
- 点击 **"清除筛选"** 恢复显示所有记录

### 6️⃣ 查看统计

页面顶部的统计卡片实时显示：
- 📊 **总记录数**：所有账单记录的数量
- 💰 **总费用**：所有记录的费用总和
- 💧 **总水费**：所有记录的水费总和
- ⚡ **总电费**：所有记录的电费总和

### 7️⃣ 快速计算器

如果只是想快速计算费用而不保存记录：

1. 点击 **"🧮 快速计算"** 按钮
2. 输入：
   - 用水量（吨）
   - 水单价
   - 用电量（度）
   - 电单价
   - 管理费
3. 系统实时显示计算结果

## 计算公式说明

### 用量计算
```
用水量 = 本月水表读数 - 上月水表读数 + 水表补差
用电量 = 本月电表读数 - 上月电表读数 + 电表补差
```

### 费用计算
```
水费 = 用水量 × 水单价
电费 = 用电量 × 电单价
总费用 = 水费 + 电费 + 管理费
```

### 示例

假设：
- 本月水表：2893，上月水表：2870，补差：1
- 本月电表：7660，上月电表：7264，补差：10
- 水价：4.3 元/吨，电价：0.72 元/度
- 管理费：22 元

计算过程：
```
用水量 = 2893 - 2870 + 1 = 24 吨
用电量 = 7660 - 7264 + 10 = 406 度

水费 = 24 × 4.3 = 103.2 元
电费 = 406 × 0.72 = 292.32 元
总费用 = 103.2 + 292.32 + 22 = 417.52 元
```

## 数据存储

- 所有数据存储在 `data/billing_records.json` 文件中
- 数据自动保存，无需手动保存
- 可以备份该文件以保留历史数据
- 删除该文件将清空所有数据

## 常见问题

### Q1：如何备份数据？
**A**：复制 `data/billing_records.json` 文件到安全位置。

### Q2：如何恢复数据？
**A**：将备份的 `billing_records.json` 文件复制到 `data/` 目录下。

### Q3：如何更改端口？
**A**：编辑 `main.go` 文件，将 `:8080` 改为其他端口，如 `:3000`，同时修改 `static/app.js` 中的 `API_BASE_URL`。

### Q4：如何在局域网访问？
**A**：
1. 启动服务器
2. 查看本机IP地址（如 192.168.1.100）
3. 其他设备访问 `http://192.168.1.100:8080`
4. 确保防火墙允许端口 8080

### Q5：服务器无法启动怎么办？
**A**：
1. 检查 Go 是否正确安装：`go version`
2. 检查端口 8080 是否被占用
3. 查看错误日志信息

### Q6：页面显示不正常？
**A**：
1. 清除浏览器缓存
2. 使用 Chrome、Firefox、Edge 等现代浏览器
3. 按 F12 查看控制台错误信息
