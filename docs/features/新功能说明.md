# 新功能说明

## 1. 房号排序功能

### API 使用

#### 获取所有记录并排序
```bash
# 按房号升序排序
GET /api/v1/billing?sortBy=room&order=asc

# 按房号降序排序
GET /api/v1/billing?sortBy=room&order=desc
```

#### 生成报表并排序
```bash
# 按房号升序生成报表
GET /api/v1/billing/report/generate?month=2025-11&sortBy=room&order=asc

# 按房号降序生成报表
GET /api/v1/billing/report/generate?ids=1,2,3&sortBy=room&order=desc
```

### PowerShell 示例
```powershell
# 获取所有记录，按房号升序
Invoke-RestMethod -Uri "http://localhost:8080/api/v1/billing?sortBy=room&order=asc"

# 生成报表，按房号降序
Invoke-RestMethod -Uri "http://localhost:8080/api/v1/billing/report/generate?month=2025-11&sortBy=room&order=desc"
```

## 2. 自定义额外费用功能

### 数据结构

每个账单记录可以包含多个额外费用项：

```json
{
  "roomNumber": "101",
  "currentWater": 2893,
  "previousWater": 2870,
  "waterPrice": 4.3,
  "currentElectric": 7660,
  "previousElectric": 7264,
  "electricPrice": 0.72,
  "managementFee": 22,
  "billingMonth": "2025-11",
  "extraFees": [
    {
      "name": "水管维修费",
      "amount": 50.00
    },
    {
      "name": "垃圾清理费",
      "amount": 20.00
    }
  ]
}
```

### 特性

- ✅ **灵活添加**：可以添加任意数量的额外费用项
- ✅ **自定义名称**：费用名称完全自定义（如"水管维修费"、"公共区域维护"等）
- ✅ **自动计算**：额外费用会自动计入总费用
- ✅ **条件显示**：只有当存在额外费用时才在图片中显示
- ✅ **动态布局**：图片高度自动根据额外费用数量调整

### API 示例

#### 创建带额外费用的记录

```powershell
$data = @{
    roomNumber = "101"
    currentWater = 2893
    previousWater = 2870
    waterAdjustment = 1
    waterPrice = 4.3
    currentElectric = 7660
    previousElectric = 7264
    electricAdjustment = 10
    electricPrice = 0.72
    managementFee = 22
    billingMonth = "2025-11"
    extraFees = @(
        @{
            name = "水管维修费"
            amount = 50.00
        },
        @{
            name = "公共区域清洁费"
            amount = 30.00
        }
    )
} | ConvertTo-Json -Depth 10

Invoke-RestMethod -Uri "http://localhost:8080/api/v1/billing" `
    -Method POST `
    -ContentType "application/json" `
    -Body $data
```

#### 更新记录添加额外费用

```powershell
$data = @{
    id = 1
    roomNumber = "101"
    currentWater = 2893
    previousWater = 2870
    waterAdjustment = 1
    waterPrice = 4.3
    currentElectric = 7660
    previousElectric = 7264
    electricAdjustment = 10
    electricPrice = 0.72
    managementFee = 22
    billingMonth = "2025-11"
    extraFees = @(
        @{
            name = "电梯维修费"
            amount = 100.00
        }
    )
} | ConvertTo-Json -Depth 10

Invoke-RestMethod -Uri "http://localhost:8080/api/v1/billing/1" `
    -Method PUT `
    -ContentType "application/json" `
    -Body $data
```

### 图片显示效果

在生成的账单图片中：

1. **【其他费用】** 区域会包含：
   - 管理费（始终显示）
   - 额外费用列表（如果有的话）
     - 水管维修费: ¥50.00
     - 垃圾清理费: ¥20.00
     - 等等...

2. **总费用** 会自动包含所有额外费用

3. **动态高度**：
   - 无额外费用：标准高度
   - 有额外费用：高度自动增加

## 常见使用场景

### 场景1：维修费用
```json
"extraFees": [
  {"name": "水管维修费", "amount": 150.00},
  {"name": "电路检修费", "amount": 80.00}
]
```

### 场景2：公共设施费用
```json
"extraFees": [
  {"name": "公共区域清洁", "amount": 25.00},
  {"name": "电梯维护费", "amount": 35.00},
  {"name": "监控系统维护", "amount": 20.00}
]
```

### 场景3：临时费用
```json
"extraFees": [
  {"name": "停车位租金", "amount": 200.00},
  {"name": "垃圾清运费", "amount": 15.00}
]
```

---

## 注意事项

1. **JSON 格式**：`extraFees` 字段可以为空数组 `[]` 或完全省略
2. **费用计算**：系统会自动计算所有额外费用的总和并加入总费用
3. **名称长度**：建议费用名称不超过10个字符，以保证图片显示美观
4. **金额精度**：支持小数点后两位
5. **排序功能**：当前仅支持按房号排序，未来可扩展其他排序方式

## 完整示例

```powershell
# 1. 创建带额外费用的记录
$newRecord = @{
    roomNumber = "203"
    currentWater = 3100
    previousWater = 3050
    waterAdjustment = 2
    waterPrice = 4.3
    currentElectric = 8200
    previousElectric = 7900
    electricAdjustment = 5
    electricPrice = 0.72
    managementFee = 22
    billingMonth = "2025-11"
    extraFees = @(
        @{name = "水管维修费"; amount = 120.00},
        @{name = "公共清洁费"; amount = 30.00}
    )
} | ConvertTo-Json -Depth 10

$result = Invoke-RestMethod -Uri "http://localhost:8080/api/v1/billing" `
    -Method POST `
    -ContentType "application/json" `
    -Body $newRecord

# 2. 查看所有记录，按房号升序
$allRecords = Invoke-RestMethod -Uri "http://localhost:8080/api/v1/billing?sortBy=room&order=asc"

# 3. 生成排序后的报表
$report = Invoke-RestMethod -Uri "http://localhost:8080/api/v1/billing/report/generate?month=2025-11&sortBy=room&order=asc"
Write-Host "报表已生成：$($report.data.filename)"
```
