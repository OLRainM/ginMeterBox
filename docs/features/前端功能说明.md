# 前端新功能说明

## ✅ 已添加的功能

### 1. 房号排序功能

#### 使用位置
在控制面板的"筛选区域"下方，新增了"排序"区域

#### 功能按钮
- 🔼 **房号升序** - 按房号从小到大排序（如 101 → 102 → 201 → A1 → A2）
- 🔽 **房号降序** - 按房号从大到小排序（如 A2 → A1 → 201 → 102 → 101）

#### 功能特点
- ✅ 点击排序按钮后，表格数据实时排序
- ✅ 排序状态持久化：创建/编辑记录后保持当前排序
- ✅ 生成报表时自动应用当前排序
- ✅ 提示消息显示当前排序状态

#### 使用流程
```
1. 点击"房号升序"或"房号降序"按钮
2. 表格数据自动按选定顺序排列
3. 后续的报表生成也会按此顺序排列
```

### 2. 额外费用管理功能

#### 使用位置
在添加/编辑记录的表单中，管理费下方

#### 功能区域
- **💵 额外费用** 区域
- 提示文字："可添加水管维修、公共清洁等额外费用项"
- **➕ 添加额外费用** 按钮

#### 功能特点
- ✅ 动态添加多个额外费用项
- ✅ 每项包含：费用名称 + 金额
- ✅ 可随时删除不需要的费用项
- ✅ 自动计入总费用
- ✅ 表格中显示额外费用总额和项数
- ✅ 鼠标悬停显示费用明细

#### 数据表格显示
| 列名 | 显示内容 | 示例 |
|------|---------|------|
| 额外费用 | 无额外费用时显示 "-" | - |
| 额外费用 | 有额外费用时显示总额和项数 | ¥195.50 (3项) |
| 鼠标悬停 | 显示所有费用名称 | 水管维修费, 公共清洁费, 电梯维护费 |

#### 使用流程
```
【添加额外费用】
1. 点击"新增记录"或"编辑"按钮
2. 填写基本水电信息
3. 滚动到"额外费用"区域
4. 点击"➕ 添加额外费用"按钮
5. 输入费用名称（如"水管维修费"）
6. 输入金额（如 120.50）
7. 可继续点击添加更多费用项
8. 不需要的费用项可点击 ✕ 删除
9. 点击"保存"按钮

【查看额外费用】
- 表格中"额外费用"列显示总额
- 鼠标悬停在金额上可查看明细
```

## 🔧 技术实现

### JavaScript全局变量
```javascript
let currentSortOrder = null;  // 当前排序顺序 ('asc' 或 'desc')
let extraFeeCounter = 0;      // 额外费用输入框计数器
```

### 关键函数

#### 排序相关
- `sortByRoom(order)` - 按房号排序
- `loadRecords(sortOrder)` - 加载记录（支持排序参数）

#### 额外费用相关
- `addExtraFeeInput(name, amount)` - 添加额外费用输入框
- `removeExtraFeeInput(id)` - 删除额外费用输入框
- `clearExtraFeeInputs()` - 清空所有额外费用输入框
- `getExtraFees()` - 获取所有额外费用数据
- `loadExtraFees(extraFees)` - 加载额外费用到表单

### API集成
```javascript
// 创建/更新记录时包含额外费用
const data = {
  ...基本字段,
  extraFees: getExtraFees()  // 自动收集额外费用
};

// 生成报表时应用排序
let url = `${API_BASE_URL}/billing/report/generate?month=${month}`;
if (currentSortOrder) {
  url += `&sortBy=room&order=${currentSortOrder}`;
}
```

## 📝 注意事项

1. **排序状态**
   - 刷新页面后排序状态会重置
   - 切换月份筛选不会清除排序
   - 清除筛选不会清除排序

2. **额外费用**
   - 费用名称建议不超过10个字符
   - 金额支持两位小数
   - 空白或金额为0的费用项不会保存
   - 关闭表单会自动清空未保存的额外费用

3. **兼容性**
   - 旧数据（无额外费用）正常显示
   - 新功能不影响现有功能
   - 所有浏览器正常工作
