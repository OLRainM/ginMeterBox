## 🗑️ 批量删除功能

### 功能位置
控制面板 > 批量操作区域 > 🗑️ 批量删除按钮

### 使用方法

#### 操作步骤
1. **选择要删除的记录**
   - 在表格中勾选需要删除的记录复选框
   - 可使用表头全选复选框快速选择所有记录

2. **点击批量删除按钮**
   - 点击"🗑️ 批量删除"按钮

3. **确认删除**
   - 系统弹出确认对话框，显示将要删除的记录数量
   - 警告提示"⚠️ 此操作不可撤销！"

4. **完成删除**
   - 点击"确定"执行删除
   - 系统显示删除成功的记录数量
   - 自动刷新数据并清除选择状态

### 功能特点

- ✅ **批量操作** - 一次性删除多条记录
- ✅ **安全确认** - 二次确认避免误操作
- ✅ **明确提示** - 显示将要删除的记录数量
- ✅ **不可撤销警告** - 明确告知操作不可逆
- ✅ **自动清理** - 删除后自动清除选择状态
- ✅ **状态保持** - 保持当前排序状态
- ✅ **实时反馈** - 显示删除结果统计

## 🔌 API 接口

### 批量删除接口

**端点：** `POST /api/v1/billing/batch-delete`

**请求体：**
```json
{
  "ids": [1, 2, 3, 4, 5]
}
```

**参数说明：**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| ids | array | 是 | 要删除的记录ID列表 |

**成功响应：**
```json
{
  "success": true,
  "count": 5,
  "message": "成功删除 5 条记录"
}
```

**失败响应：**
```json
{
  "success": false,
  "error": "请选择要删除的记录"
}
```

**部分失败响应：**
```json
{
  "success": true,
  "count": 3,
  "message": "成功删除 3 条记录"
}
```
*注：部分记录不存在时会跳过，只删除存在的记录*

## ⚠️ 注意事项

### 批量删除

1. **不可撤销**
   - 删除操作无法撤销
   - 建议删除前先导出备份
   - 确认对话框会明确提示

2. **数据验证**
   - 至少选择一条记录
   - 系统会跳过不存在的记录
   - 显示实际删除数量

3. **关联影响**
   - 删除记录后相关报表失效
   - 历史统计数据会改变
   - 建议谨慎操作

4. **权限控制**
   - 当前版本无权限限制
   - 所有用户都可删除
   - 建议增加权限验证（未来版本）

### 样式优化

1. **浏览器兼容**
   - 现代浏览器全支持
   - IE不支持自定义select样式
   - 移动端显示正常

2. **SVG箭头**
   - 使用内联SVG数据URI
   - 无需额外图片文件
   - 颜色可通过CSS调整

## 🔧 技术实现

### 前端实现

```javascript
// 批量删除
async function batchDeleteRecords() {
    const ids = getSelectedIds();
    
    if (ids.length === 0) {
        showNotification('请先选择要删除的记录', 'error');
        return;
    }
    
    const confirmMsg = `确定要删除选中的 ${ids.length} 条记录吗？\n\n⚠️ 此操作不可撤销！`;
    if (!confirm(confirmMsg)) {
        return;
    }
    
    const response = await fetch('/api/v1/billing/batch-delete', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ ids })
    });
    
    const result = await response.json();
    if (result.success) {
        showNotification(`成功删除 ${result.count} 条记录`, 'success');
        // 清除选择并刷新
        document.getElementById('selectAll').checked = false;
        loadRecords(currentSortOrder);
    }
}
```

### 后端实现

```go
// BatchDelete 批量删除记录
func (h *BillingHandler) BatchDelete(c *gin.Context) {
    var req struct {
        IDs []int `json:"ids"`
    }
    
    if err := c.ShouldBindJSON(&req); err != nil {
        c.JSON(http.StatusBadRequest, gin.H{
            "success": false,
            "error":   "无效的请求格式",
        })
        return
    }
    
    successCount := 0
    for _, id := range req.IDs {
        if err := h.storage.Delete(id); err == nil {
            successCount++
        }
    }
    
    c.JSON(http.StatusOK, gin.H{
        "success": true,
        "count":   successCount,
        "message": fmt.Sprintf("成功删除 %d 条记录", successCount),
    })
}
```
