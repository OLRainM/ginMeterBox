# Excel导出功能说明

## 📅 更新日期
2025年11月29日

---

## 🎯 功能概述

新增选中记录导出为Excel表格功能，支持批量导出账单数据到Excel文件，方便数据分析、打印和归档。

---

## 📊 功能特点

### 核心特性
- ✅ **批量导出** - 支持选中多条记录一次性导出
- ✅ **完整数据** - 包含所有字段（水电费、额外费用等）
- ✅ **精美样式** - 表头使用品牌色，自动列宽优化
- ✅ **即时下载** - 生成后自动触发浏览器下载
- ✅ **时间戳命名** - 文件名包含导出时间，避免覆盖
- ✅ **中文表头** - 清晰的中文列名，易于理解

### Excel表格特点
- 📋 **20个数据列** - 涵盖所有账单信息
- 🎨 **紫色表头** - 使用品牌色#667EEA
- 📏 **自动列宽** - 根据内容自动调整列宽
- 💡 **白色背景** - 清晰易读
- 📝 **格式化数据** - 数值、日期格式规范

## 🔌 API接口

### 导出Excel接口

**端点：** `POST /api/v1/billing/export-excel`

**请求体：**
```json
{
  "ids": [1, 2, 3, 4, 5]
}
```

**参数说明：**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| ids | array | 是 | 要导出的记录ID列表 |

**成功响应：**
```json
{
  "success": true,
  "message": "成功导出 5 条记录",
  "file": "exports/billing_export_20251129020530.xlsx",
  "count": 5
}
```

**失败响应：**
```json
{
  "success": false,
  "error": "请选择要导出的记录"
}
```

## ⚙️ 技术细节

### Excel生成流程

```
1. 接收请求
   ↓
2. 验证ID列表
   ↓
3. 从数据库获取记录
   ↓
4. 创建Excel文件
   ↓
5. 设置表头（样式+内容）
   ↓
6. 写入数据行
   ↓
7. 调整列宽
   ↓
8. 保存文件
   ↓
9. 返回文件路径
   ↓
10. 前端触发下载
```

### 后端实现（Go）

```go
// 创建Excel文件
f := excelize.NewFile()
sheetName := "账单记录"

// 设置表头样式
headerStyle, _ := f.NewStyle(&excelize.Style{
    Font: &excelize.Font{
        Bold:  true,
        Size:  12,
        Color: "FFFFFF",
    },
    Fill: excelize.Fill{
        Type:    "pattern",
        Color:   []string{"667EEA"},  // 品牌紫色
        Pattern: 1,
    },
    Alignment: &excelize.Alignment{
        Horizontal: "center",
        Vertical:   "center",
    },
})

// 写入数据
for i, record := range records {
    row := i + 2
    f.SetCellValue(sheetName, fmt.Sprintf("A%d", row), record.ID)
    f.SetCellValue(sheetName, fmt.Sprintf("B%d", row), record.RoomNumber)
    // ... 其他列
}

// 保存文件
filename := fmt.Sprintf("exports/billing_export_%s.xlsx", 
    time.Now().Format("20060102150405"))
f.SaveAs(filename)
```

### 前端实现（JavaScript）

```javascript
async function exportToExcel() {
    const ids = getSelectedIds();
    
    // 调用API
    const response = await fetch('/api/v1/billing/export-excel', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ ids })
    });
    
    const result = await response.json();
    
    if (result.success) {
        // 触发下载
        const a = document.createElement('a');
        a.href = `${window.location.origin}/${result.file}`;
        a.download = result.file.split('/').pop();
        document.body.appendChild(a);
        a.click();
        document.body.removeChild(a);
    }
}
```

## ⚠️ 注意事项

### 使用限制

1. **选择数量**
   - 建议单次导出不超过1000条记录
   - 数据量过大可能影响生成速度
   - 可分批导出后合并

2. **文件大小**
   - Excel文件大小约为：记录数 × 1KB
   - 100条记录 ≈ 100KB
   - 1000条记录 ≈ 1MB

3. **浏览器兼容**
   - 现代浏览器全支持
   - 自动下载功能需要允许弹出窗口
   - 移动端可能需要手动下载

### 数据安全

1. **敏感信息**
   - Excel文件包含完整账单数据
   - 请妥善保管导出的文件
   - 不要上传到公共网盘

2. **文件权限**
   - exports目录已添加到.gitignore
   - 不会被提交到版本控制
   - 本地文件请定期清理

3. **数据完整性**
   - 导出的是当前时刻的数据快照
   - 后续修改不会影响已导出文件
   - 需要最新数据请重新导出

## 🔍 常见问题

### Q1: 为什么选择记录后点击导出没有反应？
**A:** 检查以下几点：
- 是否有勾选记录
- 浏览器控制台是否有错误
- 网络连接是否正常
- 服务器是否运行中

### Q2: Excel文件在哪里下载？
**A:** 
- 浏览器默认下载文件夹
- 通常是"下载"或"Downloads"文件夹
- 可以在浏览器下载记录中查看

### Q3: 可以导出所有记录吗？
**A:** 
- 可以，使用"全选"功能
- 或者不使用筛选直接全选
- 建议数据量大时分批导出

### Q4: Excel中的额外费用如何显示？
**A:** 
- 单项：直接显示"名称:¥金额"
- 多项：用分号分隔"名称1:¥金额1; 名称2:¥金额2"
- 无额外费用时单元格为空

### Q5: 可以自定义Excel样式吗？
**A:** 
- 当前版本使用固定样式
- 导出后可在Excel中手动调整
- 未来版本可能支持样式自定义
