# Go-Ele 项目结构说明

## 📁 当前项目结构

```
go-ele/
├── 📂 handlers/              # HTTP请求处理器
│   └── billing_handler.go   # 账单相关的API处理器
│
├── 📂 models/                # 数据模型
│   └── billing.go           # 账单数据结构定义
│
├── 📂 services/             # 业务服务层
│   └── image_generator.go  # 图片生成服务
│
├── 📂 storage/              # 数据存储层
│   └── storage.go          # 数据存储和持久化
│
├── 📂 static/               # 静态资源（前端）
│   ├── index.html          # 前端主页面
│   ├── app.js              # 前端JavaScript逻辑
│   └── style.css           # 前端样式
│
├── 📂 data/                 # 数据文件
│   └── billing_records.json # 账单记录数据
│
├── 📂 reports/              # 生成的报表图片

│   └── card_*.png          # 单个卡片图片
```

## 📋 重构步骤

### 阶段1：基础整理（立即执行）

1. **创建文档目录**
   ```bash
   mkdir docs
   mkdir docs/api
   mkdir docs/features
   mkdir docs/user-guide
   ```

2. **创建测试目录**
   ```bash
   mkdir tests
   mkdir tests/scripts
   mkdir tests/data
   ```

3. **移动文件**
   ```bash
   # 移动文档
   move 新功能说明.md docs/features/
   move 前端功能说明.md docs/features/
   move 批量设置额外费用说明.md docs/features/
   move 导入导出说明.md docs/features/
   
   # 移动测试脚本
   move test_*.ps1 tests/scripts/
   move test_*.json tests/data/
   ```

### 阶段2：代码重构（可选）

1. **提取配置**
   - 创建 `config` 包
   - 将硬编码的配置提取到配置文件

2. **服务层优化**
   - 将业务逻辑从 handler 提取到 service
   - 添加接口定义

3. **中间件提取**
   - CORS 中间件独立
   - 日志中间件
   - 错误处理中间件

4. **添加单元测试**
   - models 测试
   - services 测试
   - handlers 测试

## 🏗️ 架构层次

### 当前架构（三层架构）

```
┌─────────────────┐
│   Handlers      │  HTTP请求处理
├─────────────────┤
│   Services      │  业务逻辑
├─────────────────┤
│   Storage       │  数据持久化
└─────────────────┘
```

### 建议架构（DDD分层）

```
┌─────────────────────────┐
│   Presentation Layer    │  展示层（Handlers）
│   - HTTP Handlers        │
│   - Request/Response     │
├─────────────────────────┤
│   Application Layer     │  应用层（Services）
│   - Use Cases           │
│   - Business Logic      │
├─────────────────────────┤
│   Domain Layer          │  领域层（Models）
│   - Entities            │
│   - Value Objects       │
│   - Domain Services     │
├─────────────────────────┤
│   Infrastructure Layer  │  基础设施层（Storage）
│   - Repositories        │
│   - External Services   │
└─────────────────────────┘
```

## 🚀 Makefile 建议

```makefile
.PHONY: build run test clean docker

# 构建
build:
	go build -o bin/go-ele cmd/server/main.go

# 运行
run:
	go run cmd/server/main.go

# 测试
test:
	go test -v ./...

# 测试覆盖率
test-coverage:
	go test -coverprofile=coverage.out ./...
	go tool cover -html=coverage.out -o coverage.html

# 清理
clean:
	rm -rf bin/
	rm -f coverage.out coverage.html

# 代码检查
lint:
	golangci-lint run

# 格式化
fmt:
	go fmt ./...

# Docker构建
docker-build:
	docker build -t go-ele:latest .

# Docker运行
docker-run:
	docker run -p 8080:8080 go-ele:latest
```
