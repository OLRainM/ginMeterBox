# 更新日志

## [v1.1.0] - 2025-11-29

### 🎉 新增功能

#### 1. 图片报表生成 🖼️

- **单用户卡片生成**
  - 生成精美的个人账单卡片
  - 包含完整的水电费信息
  - 尺寸：600x400px
  - 支持中文字体渲染

- **多用户报表生成**
  - 按月份生成汇总报表
  - 支持选择特定用户组合
  - 包含统计卡片和详细表格
  - 尺寸：1200x800+px（根据记录数自动调整）

- **报表功能特点**
  - 渐变背景，美观大方
  - 自动计算统计数据
  - 支持PNG格式输出
  - 自动保存到reports目录

#### 2. 自动延续功能 🔄

- **从上月自动创建新记录**
  - 自动继承上月的水电表读数
  - 自动设置本月初始值
  - 自动继承管理费和价格配置

- **预览功能**
  - 创建前可预览上月数据
  - 确认数据准确后再创建
  - 避免错误操作

- **数据继承规则**
  - 本月初始读数 = 上月当前读数
  - 价格配置自动继承
  - 管理费自动继承

### 🔧 技术改进

#### 后端改进

- **新增依赖**
  - `fogleman/gg v1.3.0` - 图形绘制库
  - `golang/freetype` - 字体渲染
  - `golang.org/x/image v0.15.0` - 图片处理

- **新增服务层**
  - `services/image_generator.go` - 图片生成服务
  - 支持多种图片样式
  - 自动字体降级处理

- **存储层扩展**
  - 新增 `GetByIDs()` 方法 - 批量获取记录
  - 新增 `GetLatestByRoomNumber()` - 获取最新记录
  - 新增 `CreateFromPrevious()` - 自动延续创建

#### 前端改进

- **UI增强**
  - 新增选择框功能（支持多选）
  - 新增自动延续模态框
  - 新增图片生成按钮
  - 表格增加选择列

- **样式优化**
  - 新增按钮样式（success, report）
  - 新增info-box样式
  - 优化模态框布局

- **交互增强**
  - 全选/取消全选功能
  - 预览上月数据
  - 实时反馈通知

### 📡 API 更新

#### 新增接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/billing/report/generate` | 生成报表图片 |
| GET | `/api/v1/billing/card/:id` | 生成单个卡片 |
| GET | `/api/v1/billing/download` | 下载图片 |
| POST | `/api/v1/billing/continue` | 自动延续创建 |
| GET | `/api/v1/billing/latest/:room` | 获取最新记录 |

#### 接口增强

- 支持按ID列表查询：`?ids=1,2,3`
- 支持按月份查询：`?month=2025-11`

### ⚠️ 注意事项

#### 系统要求

- **Windows**: 需要安装中文字体（通常已预装）
- **Linux**: 需要安装字体包
  ```bash
  sudo apt-get install fonts-wqy-microhei
  ```
- **Docker**: Dockerfile需添加字体安装

#### 兼容性

- 完全向后兼容 v1.0.0
- 现有数据无需迁移
- 所有原有功能保持不变

#### 性能影响

- 图片生成时间：<1秒
- 图片大小：100-500KB
- 建议定期清理旧图片

## 📝 版本说明

### 版本号规则

采用语义化版本号：`主版本号.次版本号.修订号`

- **主版本号**: 重大架构变更或不兼容更新
- **次版本号**: 新增功能，向后兼容
- **修订号**: Bug修复，向后兼容

### 升级指南

#### 从 v1.0.0 升级到 v1.1.0

**步骤：**
1. 停止服务器
2. 备份 `data/` 目录
3. 更新代码
4. 运行 `go mod tidy`
5. 重新编译或直接运行
6. 启动服务器

**数据迁移：**
无需任何数据迁移，完全向后兼容。

**配置变更：**
无配置变更。
